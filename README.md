# Установка

- На гитхабе нажать зелёную кнопку "Code".
- В выпадайке выбрать "Download ZIP"
- Распаковать в какую-то директорию
- Открыть chrome://extensions/
- В правом верхнем углу включить "Developer mode"
- Слева появится кнопка "Load unpacked". Нажать.
- Выбрать папку, в которую были развёрнуты исходники плагина из ZIP. Нажать "Select Folder"
- Плагин добавится в Хром и заработает.
